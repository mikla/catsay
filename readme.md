### Usage
```shell
catsay 0.1.0

USAGE:
    catsay [FLAGS] [OPTIONS] [message]

FLAGS:
    -d, --dead       Make the cat appear dead
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -f, --file <catfile>    

ARGS:
    <message>    What does the cat say? [default: Meow!]
```
